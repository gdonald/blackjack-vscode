# Blackjack for VS Code

Play Blackjack directly in VS Code.

## Usage

Open the Command Palette (`Cmd+Shift+P` on macOS or `Ctrl+Shift+P` on Windows/Linux) and run:

```
Blackjack: Play
```

![https://raw.githubusercontent.com/gdonald/blackjack-vscode/refs/heads/main/imgs/vscode_command.png](https://raw.githubusercontent.com/gdonald/blackjack-vscode/refs/heads/main/imgs/vscode_command.png)

Or use the keyboard shortcut `Cmd+Shift+B` on macOS or `Ctrl+Shift+B` on Windows/Linux.

Blackjack opens in an editor tab:

![https://raw.githubusercontent.com/gdonald/blackjack-vscode/refs/heads/main/imgs/game_play.png](https://raw.githubusercontent.com/gdonald/blackjack-vscode/refs/heads/main/imgs/game_play.png)

If you run out of money, close the tab and restart it.
